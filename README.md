## 프로젝트 소개

2학년 게임프로그래밍에서 진행한 코인피하기 게임입니다.

한 학기동안 배운 내용들을 적절히 활용해 새로운 게임을 만드는 프로젝트이며,

정해진 영역 안에서 사방에서 무작위의 위치에서 무작위의 속도로 날아오는 코인을 피하고

체력 회복과 같은 아이템을 활용에 기록을 갱신하는 게임입니다.
